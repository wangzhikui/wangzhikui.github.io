# JDK配置
```
JAVA_HOME=D:\JDK1.5
classpath=.;%JAVA_HOME%\lib\dt.jar;%JAVA_HOME%\lib\tools.jar
path=;%JAVA_HOME%\bin
```
# Tomcat配置
```
TOMCAT_HOME=D:\Tomcat
path=%TOMCAT_HOME%\bin
classpath=%TOMCAT_HOME%\lib
```
